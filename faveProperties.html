The float property

The float property is used to place elements and objects in certain places on the web page. The element containing this text uses the float: left property to align it to the left side of the page.



The box-shadow property

This element also uses the box-shadow property, which adds a shadow around it. The property for this listing is set as:

box-shadow: 5px 5px 10px rgb(35, 36, 35);

The two 5px attributes define the X and Y offset of the shadow, the third attribute, 10px defines the shadow spread and the last attribute, rgb(35, 36, 35) defines the color of the shadow.

Introduction:
This site gives an overview of a few different CSS properties that are helpfull in improving the look of a website.

The padding property

The padding is used to define the gap between the outer edges of an element and the contents of the element. In the examples below, the edges of the elements represented by a grey outline and the content is the text. The examples will use the padding-left property.

Note: All <p> elements have a default 3px padding, which is set in the faveProperties.css file.

padding-left 10px

padding-left: 60px

padding-left: 10em

padding-left: 30%

The column-count property

The column-count pproperty can be used to break an element into multiple columns. The <p> element that contains this text is set up to display in 2 columns. 

        .two-column{
            padding: 10px;
            column-count: 2;
            column-width: 50%;
            max-width: 100%;
            column-rule: 1px solid lightblue;
        }

As you can see, a class called .two-column was created to house the properties that tell whichever element receiving the class to display contents in two seperate column.

The columns can also be separated by a column-rule, shown here by a thin, light blue line.

The margin property

The margin property is used to define the gap between elements. Again, the edges of the elements are defined by a grey outline. The contents of each element will describe what the margin property is doing.

margin: 5px 200px 5px 5px

margin: 5px 5px 5px 200px

margin: 5px 60% 5px 5px

margin: 5px 5px 5px 60%
