THE BEST HTML TAGS

ACCORDING TO ME
Introduction: This site was designed to describe my three favorite Hypertext Markup Language (HTML) tags. It also showcases some of the different things I've learned about alternate styling and functionality with the use of Bootstrap. Click the buttons below to display information about the tags.
The <table> tag allows for the setting up of a grid system for laying out the content of your web site. They are very flexible, letting the creator decide how large the table is, along with size of the rows, columns and cells within the table.
I like to think of the <div> tag as the master tag. It is used to apply styling to the site, and has many properties that can be assigned to it. The boxes containing this information, as well as the buttons are created with <div> tags with classes defined with Bootstrap.
Without the <button> tag, web forms and web applications (the basis of this class) wouldn't work as well as they do. Buttons are used for actions, to submit data, among many other things. On this site, I used buttons to trigger an action for an accordion affect, displaying information about my favorite tags.

Conclusion: There are many different oppinions on which html tags are best, but the tags above are among my favorite. I believe they are the core tags for laying out your site the way you want it.
